# PR-FLOATING-LABEL

### Pablo Rostirolla

## Example

```
import { FloatingInput } from 'pr-floating-input';
```

```
function App() {
  return (
    <>
      <FloatingInput label="Nombre" />
    </>
  );
}

export default App;

```

## Example of optional parameters

```
function App() {
  return (
    <>
      <FloatingInput label="Nombre" error={true} />
    </>
  );
}

export default App;

```
